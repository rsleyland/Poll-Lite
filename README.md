# Poll-Lite
Real-time poll voting system

### Features
- Create poll
- Vote in poll
- See live poll updates

### Tech
##### Frontend
- React
- React Bootstrap
- Web Sockets

##### Backend
- Django
- Django rest framwork
- Redis
- Django channels


##### To run Backend
- `cd backend`
- `create virtual env (python -m venv myenv)`
- `pip install -r requirements.txt`
- `python manage.py runserver`

##### To run Fronend
 - `cd frontend`
 - `npm install`
 - `npm start`
